# Viatach A/S - Farvepalette

Denne farvepalette er designet til at give Viatach A/S et coolt, teknologisk og troværdigt udtryk.

## Core Farver

### Baggrund (Hoved)
- **Navn:** Background Black
- **Hex:** `#000000`
- **RGB:** `0, 0, 0`
- **Anvendelse:** Skaber et luksuriøst, moderne, og teknologisk fundament. Bruges som primær baggrund på alle digitale platforme.

### Primær Farve (The Cool Blue)
- **Navn:** Viatach Blue
- **Hex:** `#70BBFF`
- **RGB:** `112, 187, 255`
- **Anvendelse:** Bruges til overskrifter, vigtige knapper, grafer og nøgleelementer. Dette er Viatachs signatur-farve og skal dominere i design.

### Accent Farve (Lime Green)
- **Navn:** Viatach Green
- **Hex:** `#9AD038`
- **RGB:** `154, 208, 56`
- **Anvendelse:** Sekundær farve til call-to-actions, advarsler/notifikationer, eller som visuelt "pop" i designet. Bruges sparsomt for maksimal effekt.

## Neutrale Farver

### Neutral Farve (Tekst)
- **Navn:** Primary Text White
- **Hex:** `#FFFFFF` / `#F0F0F0`
- **RGB:** `255, 255, 255` / `240, 240, 240`
- **Anvendelse:** Hovedfarve til brødtekst for maksimal læsbarhed mod den sorte baggrund.

### Sekundær Grå
- **Navn:** Secondary Gray
- **Hex:** `#555555`
- **RGB:** `85, 85, 85`
- **Anvendelse:** Til mindre vigtig tekst, footers, eller som kantlinjer/separatorer.

## Kontrast Note

Den sorte baggrund (`#000000`) sikrer, at både Viatach Blue (`#70BBFF`) og Viatach Green (`#9AD038`) springer i øjnene og skaber et dynamisk look.

## CSS Variable Names

Når du implementerer, brug disse CSS variable navne:

```css
--viatach-bg-black: #000000;
--viatach-blue: #70BBFF;
--viatach-green: #9AD038;
--viatach-text-white: #FFFFFF;
--viatach-text-light: #F0F0F0;
--viatach-gray: #555555;
```

## Accessibility

- **Kontrast mellem #000000 og #FFFFFF:** 21:1 (AAA)
- **Kontrast mellem #000000 og #70BBFF:** 8.59:1 (AAA)
- **Kontrast mellem #000000 og #9AD038:** 6.74:1 (AA)

Alle farvekombinatoner opfylder WCAG 2.1 AA standard for læsbarhed.
